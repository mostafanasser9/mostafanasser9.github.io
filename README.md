## Tech Stack

- **Frontend:** React, Material UI, HTML5, CSS3, JavaScript
- **Backend:** Node.js, Express, Java
- **Databases:** MongoDB, Firebase
- **Tools & Others:** Git, Docker, Figma, VS Code

- **_layouts/**: Contains layout templates for the website.
  - `default.html`: Default layout template for pages.
  - `post.html`: Layout template for individual blog posts.
  - `page.html`: Layout template for standard pages.

- **_posts/**: Contains blog posts in Markdown format.
  - `2023-06-01-welcome-post.md`: A welcome post introducing the website.

- **_projects/**: Contains project files for the notebook renderer sample.

- **_data/**: Contains data files for dynamic content generation.
  - `projects.json`: JSON data related to projects.

- **pages/**: Contains additional pages in Markdown format.
  - `about.md`: About page content.
  - `contact.md`: Contact page content.

- **.gitignore**: Specifies files and directories to be ignored by Git.

- **_config.yml**: Configuration settings for the project.

- **index.html**: Main entry point of the website.

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `index.html` in your web browser to view the website.
4. Modify the content and styles as needed to personalize your website.
