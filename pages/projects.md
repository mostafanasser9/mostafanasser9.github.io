## Project Details

### Mazika
A music streaming UI built with React (Vite) and Material UI. It features dynamic search, trending songs, artist pages, a mini player, and a clean, responsive design. Perfect for showcasing UI/UX skills and React component architecture, with a Node.js/Express backend using MongoDB Atlas and SendGrid.

**Key Features:**
- Dynamic search functionality
- Trending songs section
- Artist profile pages
- Mini player with playback controls
- Responsive design for all device sizes
- Backend API with Express
- MongoDB Atlas integration
- Email functionality with SendGrid

**Technologies Used:**
- React with Vite for fast development
- Material UI for component library
- Node.js and Express for backend API
- MongoDB Atlas for database
- SendGrid for email services
- CSS Grid and Flexbox for layouts
- React Router for navigation

### Car Zone
Android-based car marketplace app developed as part of a university course. A comprehensive platform for car listings and purchases built with JavaFX for a modern, native experience.

**Key Features:**
- User authentication and profile management
- Car listings with detailed specifications
- Video previews of cars
- Search and filter functionality
- User profiles and saved listings

**Technologies Used:**
- Java as the primary programming language
- JavaFX for UI components and design
- Android-compatible implementation
- Local database for data persistence

### Personal Finance Tracker
Java desktop app for tracking personal finances, built for my Software Component Design course. This project showcases advanced software engineering principles by implementing various design patterns.

**Key Features:**
- Expense tracking and categorization
- Income management
- Budget planning tools
- Visual reports
- Implementation of multiple design patterns

**Design Patterns Implemented:**
- **Creational Patterns:** Singleton, Builder, Prototype
- **Structural Patterns:** Decorator, Proxy, Facade, Adapter
- **Behavioral Patterns:** Command, Strategy, State

**Technologies Used:**
- Java programming language
- Swing for the desktop UI
- MVC architecture
- Object-oriented design principles
